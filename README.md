## Next.js 마이그레이션

### 1. 프로젝트 루트 디렉토리에서 아래 커맨드를 실행합니다.

node v18.X.X 이상을 사용해야 합니다.(node -v)

npm install next react react-dom
npx next dev

그래도 안되면
rm -rf node_modules package-lock.json
npm install

### 2. 로컬에서 실행시, 이래 커맨드 실행 시 나오는 url로 접속합니다.

npm run dev
